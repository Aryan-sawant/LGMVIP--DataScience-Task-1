# Iris Flowers Classification

![Iris Flowers Classification Demo]

## Overview

Welcome to the Iris Flowers Classification project! This repository contains the code and resources for a machine learning project that classifies Iris flower species using four powerful classifiers: Logistic Regression, Decision Tree, Random Forest, and k-Nearest Neighbors. The project includes data exploration, model training, and an interactive user input feature for real-time classification.

## Installation

To get started with the project, follow these installation steps:

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Aryan-sawant/LGMVIP--DataScience-Task-1.git
   ```

2. Install the required libraries using pip:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

- Run the project:
   ```bash
   python LGM Task1.ipynb
   ```

- Interact with the model by inputting flower attributes for real-time species classification.

## Results

Our project showcases the power of machine learning with four robust classifiers, achieving accurate Iris flower species classification. Experience the world of machine learning and explore the possibilities of real-time classification.
